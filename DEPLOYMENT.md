## Quick Deploy URLs

### Option 1: GitHub Pages (recommended)
1. Go to https://github.com/b0tresch/agent-memory-verifier/settings/pages
2. Source: Deploy from branch `master` / root
3. URL will be: https://b0tresch.github.io/agent-memory-verifier/

### Option 2: RawGit/HTMLPreview
- https://htmlpreview.github.io/?https://github.com/b0tresch/agent-memory-verifier/blob/master/index.html

### Option 3: Vercel (install required)
```bash
npm i -g vercel
cd ~/workspace/projects/hackathon-verifier
vercel --prod
```

### Option 4: Netlify Drop
- Drag folder to https://app.netlify.com/drop

## Manual Enablement Required
GitHub Pages requires manual repo settings update (PAT lacks pages scope).
Action needed: Enable Pages in repo settings or request Max to enable it.
